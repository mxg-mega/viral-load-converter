Viral Load Calculator v1.0.0

A desktop application for calculating viral load values for HBV, HCV, and HIV tests.

Features:
- Calculate viral load values for HBV, HCV, and HIV
- Convert results to IU/ml
- Calculate logarithmic values
- Customizable conversion constants
- User-friendly interface

Installation:
1. Run the installer
2. Follow the installation wizard
3. Launch the application from the Start menu or desktop shortcut
